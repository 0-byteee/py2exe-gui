Troubleshooting
Antivirus Warning: Some antivirus software may flag the application. This is a false positive. Add the application to your antivirus exception list.

Application Won't Start: Try running as Administrator or check if all files are extracted from the ZIP.

Conversion Fails: Ensure your Python script has no errors and all dependencies are properly imported.

Missing DLL Error: Install the latest Microsoft Visual C++ Redistributable.

Frequently Asked Questions
Q: Do I need Python installed on my computer?
A: No, this is a standalone application. No Python installation is required.

Q: Can I convert multiple files at once?
A: Currently, the application converts one file at a time for optimal stability.

Q: Where are the converted files saved?
A: In the output folder you select during the conversion process.

Q: How do I disable the background music?
A: Go to the Settings tab and uncheck "Enable Background Music".

Q: Is my Python code safe?
A: Yes, your code is processed locally and never sent to any server.
